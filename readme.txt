■機能■
  STL(バイナリ形式)を読み込み、表示します。

■操作方法■
  [STLを開く]
    右上のファイルを開くボタンを押して、STLファイルを選んでください。

■マウス操作■
  右クリックドラッグ : 回転(原点)
  左クリックドラッグ : 移動
  中クリックドラッグ : ズーム
  中ホイール         : ズーム

■Shaderについて■
  index.htmlに記載しております。
  vs_mesh : VertexShaderです。
  fs_mesh : FragmentShaderです。


■制限事項■
  Chrome, Firefoxに対応しております。safariで動作しない原因は解析中です。
  STLのロード時に時間がかかる場合があります。
  STL以外のロード時は、特に何も警告が出ません。現在ガード処理の導入実装検討中です。
  CSVデータからのグループ生成時に時間がかかる場合があります。
  CSVデータに抜けは現状許していません。
  プロパティのradiusの保存に失敗する場合があります。
  OpenCSVはCSV以外はロードできません。現在ガード処理を導入処理の検討中です。
